# Data Directory

This folder is a placeholder for dataset storage and is not included in the repository due to size constraints.

To use this project:

1. Download the datasets from:
   - **INTERNAL USAGE:**
     - BirdDataset_2025_10k (orthomosaics)
       - Google Drive: [link](https://drive.google.com/drive/folders/162iF5tBlu45Qk0rMXyElD69RuQBxy5bh?usp=drive_link)
     - BirdDataset_2025_nonoverlapping_tiles_500
       - Dropbox: [link](https://www.dropbox.com/home/BirdDataset_2025_nonoverlapping_tiles_500)
       - Google Drive: [link](https://drive.google.com/drive/folders/1TGGjAU-c7CREzCSbf4br_PAFYlpZcqAu?usp=drive_link)
   - **PUBLIC USAGE:**
     - Hugging Face: *Not yet available.* Reach out to ng53@rice.edu for access to internal links.

2. Place the dataset in this directory with the structure:

```
data/
├── BirdDataset_2025_10k/
└── BirdDataset_2025_nonoverlapping_tiles_500/
```
